# 🍔 FoodieHub – Online Food Delivery Management System

FoodieHub is a scalable and robust **Online Food Delivery Management System** developed using **Java Spring Boot**. It streamlines restaurant operations, customer ordering, payment processing, and order management through a secure RESTful API architecture.

Built with industry-standard technologies such as **Spring Boot, Hibernate, JPA, MySQL, and Maven**, the project follows a clean **Layered Architecture** to ensure maintainability, scalability, and performance.

---

## 📌 Overview

FoodieHub provides a centralized platform where customers can browse restaurants, explore menu items, place food orders, make payments, and track order status efficiently.

The system also enables restaurants to manage food items, process incoming orders, and maintain customer satisfaction through smooth digital operations.

---

## 🚀 Core Modules

### 👤 Customer Management
- Register new customers
- Update customer details
- Delete customer records
- View all customers
- Search customer by ID or contact number

### 🏪 Restaurant Management
- Add and manage restaurants
- Update restaurant information
- Search by name or location
- Pagination and sorting support
- View restaurant menu items

### 🍕 Menu Item Management
- Add food items
- Update item price
- Delete menu items
- View available items
- Search items by name
- Filter items by price

### 📦 Order Management
- Place customer orders
- Cancel orders
- Update order status
- Get customer order history
- Search orders by date, amount, status, restaurant

### 🛒 Order Item Management
- Add items to existing orders
- Update quantity
- Remove items
- View all order items

### 💳 Payment Management
- View payment details
- Search by payment status
- Search by payment method
- Update payment status
